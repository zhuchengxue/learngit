# 如何有效的背单词
最终我们希望达到的一种状态是看到一个单词，然后在脑海中出现它的意像。而这个过程是一种本能的状态。
## 具体的实现方式
### 读原版
必须准备书本以及配套的有声版，这个量一定要足够，如果太少的话，其实就是课文，或者就是会话，只有足够的量级才是最好的学习方式。
### 开始
1. 一边听朗读，一边标记不认识的单词；
2. 最好使用福熙阅读器，设置高亮快捷键，例如：“Ctrl+ l.”
3. 然后用有道词典，鼠标取词功能，把高亮的单词收到单词本
4. 将有道单词添加到有道云单词，开始在云单词里面记忆单词。这是有大块时间的情况下。
5. 可以在吃饭时候，开启背景音听记单词，开启有道词典的自动模式。

### 需要留意的
1. 发音
2. 单词的拼写
3. 利用大屏幕显示屏

### 最后
不要让背单词变成一种差事
